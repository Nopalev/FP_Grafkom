## Keys

- WASD (W: forward, S: backward, A: left, D: right)
- RF (R: elevate, F: descend)
- QE (roll)
- ↑ ↓ (pitch)
- ← →(yaw)
- ZX (Z: slows simulation, X: speeds simulation)
